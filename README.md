# QRForge — Smart QR Generator

QRForge is a modern, user-friendly QR code generator designed to be fast, simple, and visually appealing. It allows anyone to create customized QR codes for everyday use without any technical knowledge.

The goal of this project is to provide a clean and powerful tool that combines functionality with a smooth and attractive user experience.

---

## How to Use

- Open the website
- Select the type of QR you want
- Enter your content
- Customize (optional)
- Click Generate QR
- Download and use it anywhere

--- 

## 🚀 Key Features

- Generate QR codes instantly for multiple types of content
- Support for:
  - Text
  - URLs
  - Email
  - Phone numbers
  - SMS
  - Contact (vCard)
- Live QR preview
- Download QR code as an image (PNG)
- Adjustable QR size
- Custom QR color (foreground)
- Custom background color
- Clean and responsive UI (works on desktop and mobile)
- Interactive tabs for different QR types
- Simple and intuitive user interface
- No login required — open for everyone

---

## 🌟 Unique Selling Points (USP)
